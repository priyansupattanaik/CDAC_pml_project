Machine Learning Pipeline

Our model represents a robust machine learning architecture designed to handle severe class imbalances without losing prediction accuracy.

1. Class Balancing

SMOTE (Synthetic Minority Over-sampling Technique) synthesized new attrition records, increasing representation of minority classes and preventing bias.

2. Feature Scaling

A StandardScaler scaling algorithm centers all 29 numerical features to mean=0 and variance=1 inside the predictive pipeline.

3. Random Forest Classifier

An ensemble model of 100 individual decision trees trained on the scaled, resampled data. Predicts with premium mathematical certainty.
